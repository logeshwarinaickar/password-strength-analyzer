# 🔐 SecurePass – Password Strength Analyzer

SecurePass is an interactive Password Strength Analyzer and Password Generator built using Streamlit.

The app analyzes passwords in real-time and provides:
- Password strength dashboard
- Security suggestions
- Password metrics
- Secure password generator

---

## 🚀 Features

### 📊 Password Analysis
- Checks password length
- Counts uppercase letters
- Counts numbers
- Counts special characters

### 🛡️ Strength Dashboard
Displays:
- Weak score
- Moderate score
- Strong score

using progress bars.

### 💡 Suggestions
Provides security improvement suggestions such as:
- Increase password length
- Add numbers
- Include uppercase letters
- Use special characters

### 🔑 Secure Password Generator
Generates secure random passwords with:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/logeshwarinaickar/password-strength-analyzer.git
```

Go to project folder:

```bash
cd PASSWORD GENERATOR
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run password.py
```
